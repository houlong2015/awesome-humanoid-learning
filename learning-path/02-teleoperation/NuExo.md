# NuExo: A Wearable Exoskeleton Covering all Upper Limb ROM for Outdoor Data Collection and Teleoperation of Humanoid Robots

> 论文：Rui Zhong, Chuang Cheng, Junpeng Xu, Yantong Wei, Ce Guo, Daoxun Zhang, Wei Dai*, Huimin Lu*  
> 机构：国防科技大学  
> arXiv：[2503.10554](https://arxiv.org/abs/2503.10554)（2025-03）  
> 标签：`Teleoperation` · 可穿戴外骨骼 · 户外多模态采集

---

## 一、总结

NuExo 是一套**背包式可穿戴上肢外骨骼**，面向人形机器人的**户外数据采集与遥操作**。核心创新是**联动 + 同步带肩部机构**：在减少驱动电机的同时补偿胸锁/盂肱关节（GH）解剖运动，实现接近 **100% 自然上肢活动度（ROM）**，整机约 **5.2 kg**，可日常户外背负使用。

系统统一提供：多模态传感（关节、力/力矩、第一人称视觉、手指、里程计等）、免标定跨平台遥操作框架，并在多台人形机器人、多名未经训练操作者上验证了活动度、长期精度与任务能力（如拧 2.5 mm 螺钉头、投球入 1.2 m 外箱等）。

相对 IMC（惯性动捕）：剧烈运动后 NuExo 静态误差仍低（约 <0.14 rad），IMC 可恶化到约 0.41 rad——刚性连杆编码不易漂移。

## 二、问题背景

从动捕/遥操作走向技能学习是具身智能关键路径，但现有设备难以同时满足：

| 目标 | 含义 |
|:-----|:-----|
| Accuracy | 长时间准确跟踪全上肢 |
| Comfort | 符合人体生物力学 |
| Versatility | 多模态（尤其力）+ 兼容多人形 |
| Convenience | 轻便，可户外日常使用 |

常见方案短板：手持/手套数据丰富性有限；VR/XR 沉浸但场景受限；视觉动捕场景与成本受限；IMU 动捕易漂移；同构座舱（Aloha/Gello）或 AirExo 等外骨骼往往缺力觉、或不可户外背包部署。康复外骨骼（如 Harmony）证明肩部解剖补偿重要，但额外补偿电机导致肩部过重、难轻量化。

## 三、系统组成

见图 2，主要模块：

1. **上肢外骨骼本体**：肩、肘、腕 + 力反馈数据手套（Dexmo）  
2. **被控人形机器人**  
3. **视觉显示与采集**：Rayneo Air2 AR 眼镜显示机器人 FPV；可选 Intel D435 RGB-D 第一人称上肢视频  
4. **控制与供电**：Intel N5105 等；急停；基座 IMU 里程计  

绑定处装 **六维力传感器（KunWei）**，采集人-外骨骼交互力——多数采集系统不具备。腕部用**差分 IMU** 替代复杂机构，便于接手套。总重 **5.2 kg**，主材料碳纤维板 + 3D 打印 PLA。

可同时采集：

1. 机器人遥操作数据  
2. 人上肢运动学  
3. 第一人称运动视频  
4. 手指运动  
5. 里程计  
6. 上肢力反馈  

作者称其为人形上肢研究中**最全面的多模态户外可移动**采集平台之一。

## 四、方法详解

### 4.1 肩部机构：GH 中心动态跟踪

人肩可近似三自由度球铰，几何中心在盂肱（GH）关节，但相对躯干会移动。多数外骨骼忽略该位移 → 中心错位 → 不舒适、ROM 受限。

Harmony / ANYexo 等用额外自由度补偿 GH 水平与垂直位移，但肩部电机多、易干涉、难背包化。

**NuExo 关键观察**：上臂自然旋转时，GH 水平位移与某一肩球铰自由度存在映射，可用**机械传动把两自由度合为一**，减电机且保持补偿。

- **水平面**（图 3）：连杆 + 同步带/同步轮（参数见表 I，如 \(L_1=150\) mm，\(L_2=187\) mm，\(\theta_2=1.444\theta_1+0.938\)）跟踪 GH 水平轨迹。  
- **垂直方向**（图 4）：螺旋齿轮等结构跟踪 GH 升降。  

从而在**少一个驱动**的前提下保持解剖补偿，首次做成轻量背包式解剖补偿外骨骼。

### 4.2 肘部结构

将原肩部一颗电机**挪到肘侧**（图 6），避免手臂上举时电机撞头；肩上仅保留一颗主动电机。配合连杆被动自由度（DH 参数表 II），获得很大 ROM。

### 4.3 统一遥操作控制

不走「仅末端位姿 + 定制主手」的窄路径，而是对齐关键量：

- 肩姿态（肱骨方向）  
- 肘关节角  
- 腕姿态  
- 手指约 6 DoF  

核心思想：**速度（twist）跟踪 + 位姿闭环消误差**，经阻抗控制输出机器人关节力矩指令；并把人-外骨骼交互力 \(F_{ft}\) 耦合进控制以提升灵敏度。

肩部示例：由 SCFK（肩胛-肩协调正运动学）得四元数位姿 \(q_m\) 与角速度；与机器人侧 \(q_s,\dot q_s\) 差分后进阻抗：

\[
q_t=q_s^* q_m,\quad \dot q_t=\hat{\dot q}_m-\hat{\dot q}_s
\]

\[
\tau^s_{cmd}=k_p f_P^{-1}(q_t)+k_d J^{-1}\dot q_t+\lambda J^\top F_{ft}
\]

肘/指可用标量关节形式。外骨骼侧：动力学补偿 + 全臂协调（FCM）与绑定对齐（BAS），实现「浮动辅助」柔顺运动。

软件四节点：外骨骼 / 人形 / 遥操作控制器 / 底层驱动——换底层驱动即可换机器人；开多节点可同时控多机。

## 五、实验与结果

### 5.1 ROM

佩戴后肩部约：屈伸 180°/60°，内收外展 30°/150°，水平屈伸 30°/135°；可轻松举过头顶、圆周运动。作者称超越 Harmony、ANYexo 等先前外骨骼的佩戴 ROM 表现。

### 5.2 与 IMC（Noitom PN Studio）精度对比

流程：标定 → 标准肩部动作基线 → 10 分钟高强度户外扰动（跑跳等）→ 再测同一动作。

- 初始双方静态误差约 0.1 rad 量级。  
- 扰动后 IMC 峰值静态误差可达约 **0.41 rad**；NuExo 维持约 **<0.14 rad**。  
- IMC 抖动更大（分布式 IMU 噪声经 FK 累积）；NuExo 刚性编码更稳。  

适合「野外长期」采集。

### 5.3 遥操作表现

- 高速跟踪：肩平均角误差约 **0.015 rad**，峰值约 0.05–0.08 rad（多因从机惯性）。  
- 慢速：平均约 **0.01 rad**；滤波器抑制 <0.015 rad 的人手颤动。  
- 力传感器可采人-外骨骼交互力；ZED 等可向 AR 眼镜推送立体 FPV 并录第一人称数据。  
- 精细：用电批拧 **M3 / 2.5 mm 螺钉头**，手指可控开关。  
- 动态：投球入约 **1.2 m** 外箱。  
- 跨人跨机：不同操作者、不同机器人完成同一拧瓶盖任务，**无需预训练**即可上手。

## 六、贡献与展望

**贡献**：

1. 联动-同步带协同肩机构：首次背包化实现胸骨锁骨/肩解剖补偿，全上肢 ROM。  
2. 多模态 + 免标定统一控制框架，跨人形零样本操作。  
3. 多平台实验验证活动度、长期精度与任务表现。

**展望**：探索多类数据（尤其力）在人形技能学习中的有效组合方式。

## 七、学习要点（结合本阶段）

- 数据链路：**人 → 可穿戴外骨骼（编码器+力+视觉+手套）→ 统一映射控制器 → 人形 → 录制**。  
- 「舒适/解剖对齐」不是体验问题，而是**长时数据质量与户外可用性**问题；相对 IMU 动捕的抗漂移是硬指标。  
- 与 HOMIE 对比：NuExo 强调**穿戴、户外、力觉、跨平台姿态跟踪**；HOMIE 强调**同构关节匹配座舱 + 下肢 RL 蹲走**。  
- 与 OmniH2O/TWIST：NuExo 偏**上肢硬件与采集完整性**；全身平衡/下肢策略需另接机器人侧控制栈。

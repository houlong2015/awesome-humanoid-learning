## 一、总结

TWIST（Teleoperated Whole-Body Imitation System）把人形遥操作建模为**实时全身运动重定向 + 跟踪**：用 MoCap 采人全身动作，重定向为机器人关节与根速度目标，再用**单一神经网络控制器**在仿真中用 RL+BC 训出可部署策略，实现真机上前所未有的**协调全身技能**。

在 Unitree G1（29 DoF）上覆盖四类能力：

| 类型 | 例子 |
|:-----|:-----|
| 全身操作 | 蹲下抬箱、扶正垃圾桶、递玩具 |
| 腿部操作 | 手持物时脚踢开门、大力踢球、脚运箱 |
| 运动 | 侧步、后退、蹲身钻障碍 |
| 表现力 | 拳击、与人共舞华尔兹步伐 |

相对 Mobile-TV / HOMIE（上下肢解耦）以及 HumanPlus（相机估姿）、OmniH2O（VR 稀疏上肢）：TWIST 强调**完整全身目标 + 统一跟踪策略**，蹲抬、踢球等协调动作显著更强。

---

## 二、问题背景

人类日常技能天然是全身协调的（手占满时用脚开门、弯腰清床底、单脚踢球等）。仅模仿离线人体数据不足以支撑具身视动控制，需要能采**机器人观测-动作**的全身遥操作系统。

难点长期存在：

- 经典模块化控制：遥操作与平衡分离 → 全身能力与鲁棒性受限  
- 学习控制器进步大，但仍缺：**准确实时的全身跟踪目标** + **能跟多样实时动作的鲁棒控制器**

## 四、方法详解

### 4.1 数据集策划

- **大规模公开**：AMASS + OMOMO，约 **15k clips / 42 h**，滤掉爬楼等不可行动作；离线重定向（类 PHC，优化关键点位置 + 时间平滑）。  
- **小规模自采**：约 **150 clips / 0.5 h**，用与真机相同的在线 MoCap + 在线重定向；动作随机，**不针对遥操作任务设计**。  

关键发现：仅加入少量「脏」的在线分布数据，未见过测试动作与真机遥操作稳定性就**显著提升**——弥合干净离线数据与实时噪声、标定漂移、遮挡、在线 IK 不平滑之间的 gap。

**在线重定向改进**：在快速 IK 式重定向中，**同时优化 3D 关节位置与朝向**（相对只优化朝向），缩小 offline/online 质量差，降低跟踪误差。

跟踪目标选**机器人局部坐标系下的关节位置 + 根速度**（非世界系），减轻累积误差与重定向不完美，并与真机设置一致。

### 4.2 两阶段控制器训练

#### 教师 \(\pi_{tea}\)

- 输入含约 **2 秒未来参考运动**，可预见目标 → 步态更平滑  
- PPO + 跟踪奖励（关键点/关节位姿速度、根位姿速度等）+ 惩罚（脚滑、接触、动作变化率等）+ 脚空中时间奖励  

#### 学生 \(\pi_{stu}\)

部署时无未来帧，观测差距大，纯 DAgger 不够。采用 **RL + BC（KL 蒸馏）**：

\[
\mathcal{L}(\pi_{stu})=\mathcal{L}_{RL}(\pi_{stu})+\lambda\,D_{KL}(\pi_{stu}\parallel\pi_{tea})
\]

\(\lambda\) 训练中逐渐减小。结果：**RL+BC ≫ 纯 RL ≫ 纯 BC(DAgger)**——RL 泛化更好，BC 带来平滑与对教师行为的模仿；纯 RL 易脚滑、全局误差大；DAgger 对未见 MoCap 偶发失稳。

#### 力相关鲁棒性

纯位置跟踪对「抬重箱」等施力任务属 OOD，易抖。训练时对末端施加较大扰动（表 2：Push End-Effector 约 0–20 N 等域随机），显著改善接触与静止持物稳定性。

域随机还包括：底座质量、摩擦、电机强度、重力微扰、推基座等，支撑 zero-shot 真机。

### 4.3 真机部署

OptiTrack 流 → 在线重定向 → \(\pi_{stu}\)（RTX 4090）出关节目标。亦可迁移到 Booster T1（IsaacGym 训、MuJoCo 评：摆臂、深蹲、走等）。

## 五、实验与分析

### 5.1 主结果

G1 上上述四类技能均由**同一网络**完成。系统分析：

- **可达性**：全身 DoF 使手几乎可够到脚尖，优于先前稀疏上肢/解耦系统。  
- **延迟**：端到端约 **0.9 s**（重定向目标生成约 0.7 s，策略约 0.2 s）——未来重点优化。  
- **跟踪误差分布**：末端（手、脚）误差最大；下肢（脚、膝）整体难于上肢——接触动力学更复杂。  
- **失败主因**：硬件过热（蹲姿任务约 5–10 分钟需冷却）；策略在极端测试下仍能努力保平衡。

### 5.2 消融结论（自采未见 50 clips 测试集）

1. **RL+BC** 跟踪精度与平滑性最优  
2. **In-house MoCap** 对未见动作与真机至关重要  
3. **末端扰动** → 持物/接触任务更稳  
4. **更好在线重定向** → 整体跟踪误差下降

## 六、局限

1. **无机器人反馈给操作者**：无自视视觉回传（遮挡时难操作）；无触觉告知抓取是否成功。  
2. **硬件可靠性**：长时间连续工作受限。  
3. **依赖 MoCap**：难普及；未来希望用 RGB 估姿逼近 MoCap 质量。  

未来方向：结合 TWIST 遥操作数据与第一人称人类数据学视动策略。

---

## 七、学习要点（结合本阶段）

- 数据链路：**人全身 MoCap → 实时重定向 → 单一跟踪策略 → 机器人 PD**；采数质量高度依赖**全身目标精度**。  
- 「人-机具身差异」靠**重定向**显式处理；「延迟」主要卡在重定向（0.7 s），不只是网络推理。  
- 训练技巧可迁移：教师看未来、学生部署；少量真实遥操作分布数据；施力任务要扰动训练。  
- 阶段地图位置：若目标是**蹲抬、踢球、钻障**等协调全身数据，TWIST 路线最贴；若强调低成本座舱精度采操作数据，看 HOMIE/NuExo；若强调 VR 通用接口与开源 LfD 数据，看 OmniH2O。
